<h1 style="color:#2082AA;">Sistema de banco de dados - Mercadinho</h3>

**Projeto academico - Faculdade cruzeiro do sul**

✨ O Mercadinho é um projeto simples e funcional desenvolvido para simular o banco de dados de um mercado de pequeno porte. Ele foi criado com foco em organização, boa modelagem e aplicação prática dos conceitos de SQL, incluindo criação de tabelas, inserção de dados, consultas, atualizações e remoções.

✨ Objetivo do Projeto

Fornecer um banco de dados totalmente funcional, capaz de:

Registrar clientes, funcionários, fornecedores e produtos

Controlar estoque

Registrar vendas e itens vendidos

Realizar consultas com JOIN, WHERE, ORDER BY, LIMIT

Executar operações de INSERT, UPDATE e DELETE
